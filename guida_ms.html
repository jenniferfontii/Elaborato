Guida Microsoft Server

Per hostare un sito sul web server è necessario seguire questi passaggi:

  • Creare una cartella contenente i file sorgenti del sito
  • Attraverso l’interfaccia di IIS selezionare l’opzione che permette il censimento di un nuovo sito web:
  • Valorizzare i vari campi richiesti dall’interfaccia.
  • Assegnando i nomi dei siti e di dominio in maniera univoca ed indicando per ciascuno lo stesso IP e la stessa porta l’interfaccia agirà utilizzando la tecnica name based
